# LMMS
Local medicine management system  
本地使用的医药管理系统

---
## 说明
基于QT/C++编写，数据库采用SQLite。  
+ 药品入库
+ 药品出库（处方）
+ 数据维护（直接修改数据）
+ 输入联想提示
+ 用户输入合法性判断（基于正则表达式）
+ 打印指定/所有药品数据至PDF文件
